So, what is all this?

I modified a fridge to grow mushrooms in it. It was fun. This site shows the current temperature and humidity in the fridge, it contains documentation on how I built everything and how you might be able to as well.

Cooking is something that brings me a lot of joy, so does rasing plants. Being able to grow my own food has been a very nice experience and I hope some people who read this will be inspired to try to do something like this themselves.

This Project was inspired by the People at the Finish Startup Helsieni.

I took part in their Mushroom growing workshop and they mentioned modifying a fridge to grow mushrooms.

This immediately intrigued me. This would allow me, someone with limited space, to grow some food for myself.

In this Blog-Post they talk about how they did a workshop helping others to build a growfridge themselves.

I decided that I wanted to expand on their project to better fit my wants and needs.

The changes I wanted to make mainly focused on the following:

1. Replacing the control components with an arduino and appropriate sensors.

2. Allowing easy control for light, humidity and temperature through a visual interface.

3. Make this project reproducible for others that are interested by making my code and setup open source.

To address the first point, I selected an arduino mega as my control board and a DHT-22 as my temperature and humidity sensor.

The sensor and internal counters in the arduino are used to control a relais board, which then turns things like the fridge or the light on or off.

While you could use a pre-made humidifier to get water vapor into the fridge, I decided to build my own, using a ultrasonic fogger as the base.

Regarding point two, for the visual interface I used a 20x4 LCD display and 4 keyboard switches to change settings.

To contain the display and buttons I designed a model that can be 3D printed and be used to mount these parts.

The third was also very important to me. Having the code and some instructions others can follow will hopefully contribute a small bit to the ever growing amateur mushroom growing community.
